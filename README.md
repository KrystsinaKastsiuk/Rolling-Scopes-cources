Webpack Dev Server, Hot Module Replacement and other Webapck features were used:

	* Babel; 
	* Uglify JS Plugin; 
	* Extract Text Plugin; 
	* Inline Source Map;
	* Webpack Notifier Plugin; 
	* CSS Loader;
	* Style Loader;
	* Watch.

Commands to start project:

	* npm i 
	* npm i webpack -g
	* webpack
	* node dev-server
